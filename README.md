# だんごバス3台 — HTML発表資料 v2

このディレクトリは、15分発表向けのHTMLスライド資料です。

## ファイル構成

- `index.html`：発表本体
- `assets/css/style.css`：スライドデザイン
- `assets/js/deck.js`：キーボード操作、ノート表示
- `assets/images/`：このスレッドで共有した画像
- `assets/svg/`：追加作成した説明図
- `speaker_notes.md`：発表者ノートと改善メモ
- `manifest.json`：利用画像・生成図の一覧

## 操作

- 右矢印 / Space：次のスライド
- 左矢印：前のスライド
- N：発表者ノートの表示 / 非表示
- F：フルスクリーン
- Home / End：最初 / 最後へ
