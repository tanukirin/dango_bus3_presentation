# だんごバス3台 発表者ノート v2

## 使い方
- `index.html` をブラウザで開く。
- `←` / `→` / Space で移動。
- `N` キーで発表者ノート表示。
- `F` キーでフルスクリーン。
- 印刷またはPDF化する場合はブラウザの印刷機能を使う。

## 15分想定の話し方
1. タイトル・導入（1分）
2. だんご運転の現象と正のフィードバック（2分）
3. 寺田寅彦の100年前の洞察（1分）
4. 自作シミュレーターと3方式比較（2分）
5. スプリング制御の逆効果発見から満員保持禁止までの流れ（3分）
6. 台数・容量・乗降時間短縮というハード面の示唆（2分）
7. 乗客行動、NEC REFLEX、EV時代の折返し設計（3分）
8. 複数分野との関連と結論（1分）

## この版で反映した重点
- タイトルは「だんごバス3台」。
- 正のフィードバックループ画像は全面表示。
- 路線の模式図は削除。
- 「スプリング逆効果の発見 → AIと原因検討 → 論文発見 → 検証 → 克服」の流れをドラマとして追加。
- 寺田寅彦『電車の混雑について』を、乗客の自発的分散の伏線として追加。
- EV充電は否定的に見せず、「運行と充電を一体設計する未来」として前向きに整理。
- 複数分野との関連は、添付の総まとめ画像を使用。

## 追加するとさらに良い画像
- 実際のシミュレーター画面スクリーンショット。
- 95%待ち時間、車間RMSE、未完了者数、補正平均総所要時間の比較グラフ。
- 満員時保持禁止の前後で、平均総所要時間や未完了者数がどう変わるかのグラフ。

## 発表の中心メッセージ
バス団子運転は「小さな遅れが増幅する」システム現象。スプリング制御は有望だが、満車・車内客・停留所容量を入れなければ逆効果にもなる。高需要では台数・容量・乗降時間短縮の効果が大きく、最終的には交通工学、制御、OR、行動科学、環境、現場運用を横断する総合問題として扱う必要がある。
